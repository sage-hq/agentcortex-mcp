## üöÄ Quick Start

### 1. Get Your API Key
```bash

curl -X POST https://api.agentcortex.dev/auth/signup \
  -H "Content-Type: application/json" \
  -d '{"email":"you@company.com"}'
```

### 2. Install & Configure
```bash
# Install globally
npm install -g agentcortex

# Or run directly
npx agentcortex
```

      "env": {
